JavaScript

A programming language.

Operators

Used to assign values, compare values, perform arithmetic operations, and more.

Variables

Named values and can store any type of JavaScript value.

Variable naming conventions

Most variables and functions should start with a lower case letter, subsequent words in name should have first letter capitalized. Names should be descriptive. Global variables in browsers should be in all caps.

Functions

a block of code designed to perform a particular task.

Methods

the actions that can be performed on objects. A property containing a function definition

Arguments

the values received by the function when it is invoked.

Parameters

listed inside the parentheses () in the function definition.

JavaScript

A programming language.

Operators

Used to assign values, compare values, perform arithmetic operations, and more.

Variables

Named values and can store any type of JavaScript value.

Variable naming conventions

Most variables and functions should start with a lower case letter, subsequent words in name should have first letter capitalized. Names should be descriptive. Global variables in browsers should be in all caps.

Functions

a block of code designed to perform a particular task.

Methods

the actions that can be performed on objects. A property containing a function definition

Arguments

the values received by the function when it is invoked.

Parameters

listed inside the parentheses () in the function definition.

Return

the value produced by a function

Chaining methods

executing multiple methods on one object in a single line, w/o requiring variables to store intermediate results

Strings

data type that contains characters

Booleans

data type that contains true or false

Undefined

a primitive value, represents a value not defined or returned

NaN

not a number.

Escape

using '\' to signal the compiler to treat the following character as special

Alert

a popup window containing a message

Comments

lines of code that aren't read by the compiler

jQuery

a javascript library, containing methods to simplify complex javascript actions

Attributes

portion of HTML elements that provides information about what type that element is
